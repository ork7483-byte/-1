코딩 지식이 전혀 없어도 수정할 수 있나요? −

제공되는 매뉴얼과 영상만 보시면 코딩 지식 없이도 누구나 쉽게 수정 가능하며, 실제로 모든 고객님이 문의 없이 스스로 완료하셨을 만큼 간편합니다.

유지보수는 어떻게 되나요? +

기본적으로 제공되는 메뉴얼을 통해 직접 유지보수가 가능합니다. 추가적인 기술 지원이나 기능 개발이 필요한 경우에는 별도의 유지보수 계약을 통해 지원해 드리고 있습니다.

커스터마이징이 뭔가요? +

기존 템플릿의 디자인이나 기능을 고객님의 요구사항에 맞춰 수정하는 작업을 말합니다. 색상, 레이아웃 변경부터 새로운 페이지 추가까지 다양한 범위의 작업이 가능합니다.
